Mini-Snake - tutorial

1. instaleren 2. HTML 3. CSS 4. javascript

Wat gaan we maken ?

In deze tutorial maken we de leuke mini-game Snake. Waar het de boedling is om als slang de appels op het veld op te eten zonder jezelf of de rand te raken.

stap 1: installatie

Ga naar de officiële website van Visual Studio Code. Op de startpagina zie je meteen een knop om de versie te downloaden die geschikt is voor jouw besturingssysteem (Windows, macOS of Linux). Klik op de knop om het installatiebestand te downloaden. Zodra je dit hebt gedaan kunnen we aan de slag met programmeren!

1.1 maak een nieuwe folder

Maak een nieuwe folder met de naam snake.

1.2 Open VS Code

Open Visual Studio Code en open de nieuwe folder.

1.3 folderstructuur

Maak de mappen css, html en js, en de bestanden style.css, index.js en index.html. Gebruik de knoppen bovenaan in de verkenner. Zorg dat de bestandsnamen exact kloppen! De map- en bestandsstructuur ziet er zo uit:

1.4 Installatie Live Server

Ga naar extensions en installeer de Live Server-extensie in Visual Studio Code.

1.5 Opstarten van Live Server

Start de Live Server door onderaan op Go Live te klikken. Dit opent een nieuw browservenster met je project.
